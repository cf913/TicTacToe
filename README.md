# Tic Tac Toe

Initial implementation of a TicTacToe and Align-4 (or Connect-4) using HTML5 CSS JQuery

## Getting Started

Clone or download this repo --> open index.html in a browser.

### Prerequisites

Web Browser (Chrome recommended)
